## 🚀 Getting Started

Once the installation finishes, find the icon on your desktop or in your Start menu. Click it to open the program. The screen shows two panels. Use the Tab key to switch between the left and right panels.

### Navigation
Move through your folders by using the arrow keys on your keyboard. Press Enter to open a folder. Use the Backspace key or click the parent folder icon to go back to the previous location.

### Common Tasks
You can perform file actions using the numbered menu at the bottom of the screen.

*   View: Press F3 to look at the contents of a selected file.
*   Edit: Press F4 to open the file for changes.
*   Copy: Press F5 to move a file from one panel to the other.
*   Move: Press F6 to transfer a file permanently.
*   Make Folder: Press F7 to create a new folder in your current location.
*   Delete: Press F8 to remove files or folders.

## 🛠 Advanced Features

Far Manager offers tools to manage large sets of data. You can sort files by name, extension, size, or date. To sort files, choose the sort option from the menu bar at the top or press Ctrl and F. 

If you need to find a specific file, press Alt and F7. Enter the file name or part of the name in the search box. Press Enter to start the scan. The program highlights the files it finds in the list.

### Plugins
The software supports plugins to add new capabilities. These tools allow you to handle archive files like ZIP or RAR directly within the interface. You can view the contents of an archive just like a normal folder.

### Colors and Layout
You can change the look of the program to suit your preferences. Access the Options menu to adjust the screen colors and the font size. Save your settings to keep them every time you launch the software.
